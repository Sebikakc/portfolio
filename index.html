About This Portfolio

This is my portfolio website to communicate the knowledge and skills for front end development and modern web development techniques. I created this project as a student looking to go for junior full stack developer roles to present professionally responsive layouts, front end interfaces, coding examples, interactive JavaScript systems, Bootstrap components, and educational web content.

The project illustrates what Full Stack is, how HTML is used to organize webpages, how CSS can enhance the beauty and layout of a webpage, how Bootstrap can make web development responsive, and how JavaScript can make webpages more interactive and dynamic. This portfolio also illustrates my knowledge about UI design principles, responsive web systems, user interaction and frontend development practices.
